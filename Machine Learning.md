**Machine Learning**

Machine Learning (ML) enables computers to learn from data and make predictions or decisions automatically.

**Types of Machine Learning**

**Supervised:** Learn from labeled data (e.g., spam detection)

**Unsupervised:** Find patterns in unlabeled data (e.g., customer segmentation)

**Reinforcement:** Learn via rewards/penalties (e.g., self-driving cars)

**Uses:** Analyze large datasets , Automate decisions , Detect hidden patterns , Improve over time

**Applications**

Healthcare, Finance, E-commerce, Transportation, IT & Testing
